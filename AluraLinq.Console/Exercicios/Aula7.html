Consulta Com Paginação

Suponha que você tenha a seguinte consulta "Linq to Entities", para trazer um relatório de notas fiscais: 

            using (var contexto = new AluraTunesEntities())
            {
                var query =
                from nf in contexto.NotasFiscais
                orderby nf.NotaFiscalId
                select nf;
                foreach (var nf in query)
                {
                    Console.WriteLine("{0}\t{1}\t{2}\t{3}", 
                    nf.Numero, nf.Data, nf.Cliente, nf.Total);
                }
            }

Agora considere que esse relatório seja dividido em páginas de 10 notas fiscais. Modifique a consulta de modo a retornar somente as notas fiscais da terceira página.
